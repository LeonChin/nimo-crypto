## Getting Started

1. Clone the repo
   ```sh
   git clone git@github.com:LeonChin/nimo-crypto.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the project
   ```sh
   npm run dev
   ```
